## 🚀 Expertise & Focus

Mon objectif est de concevoir et de déployer des architectures cloud résilientes en intégrant la sécurité dès la première ligne de code (**DevSecOps**). Mon approche combine une solide logique algorithmique héritée d'Epitech et une expertise offensive acquise en CTF.

### 🔐 Cybersecurity Labs
- **Pentesting :** Préparation active de la certification **HTB Certified Penetration Testing Specialist (CPTS)**.
- **Cloud Security :** Sécurisation d'environnements **AWS** (IAM, S3, EC2) — *Cursus AWS Academy*.
- **Threat Detection :** Analyse de vulnérabilités (OWASP Top 10), durcissement système et monitoring réseau.

### ⚙️ Automatisation & SecOps
- **Infrastructure as Code :** Déploiement et isolation réseau de clusters **Kubernetes** et **Docker** via **Ansible**.
- **Hardening :** Conception de scripts de durcissement **Linux / Windows** et de configurations YAML rigoureuses.
- **CI/CD Sécurisée :** Automatisation du cycle de vie applicatif via **Jenkins** avec intégration de scans de vulnérabilités.
